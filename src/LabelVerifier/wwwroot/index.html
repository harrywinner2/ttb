Checking engine…

1 Add the label photos

Add up to 4 photos of the same product, covering every side that has text — at least one should carry the Government Warning. Any angle is fine; clear, straight-on shots read best. JPG, PNG, or WEBP.

2 What does the application say?

These values come from the application on file (in production, pre-filled from COLA). Leave any blank to simply read it off the label.

The Government Warning is always checked.

1 Add the label photos

Select up to a few hundred images at once.

2 Add expected values (optional)

Upload a CSV (one image per row) with a filename column plus any of brand_name, class_type, alcohol_content, net_contents, bottler, country — or a JSON manifest to group several images per product (any sides/angles, up to 4) using an "images": [ … ] list. Without a manifest, each image is treated as its own product. Processing runs in parallel.

Prototype for evaluation only — not an official TTB determination. Images are processed in memory and not stored.
